# dk_installation

> Info: This repo provides a guideline to install the dreamOS.

## Prerequisites:
Git clone dk_installation repo.  
```
git clone https://github.com/ppa2hc/dk_installation.git
cd dk_installation
chmod +x *
```

## Install dreamOS
```
sudo ./dk_install.sh

## Uninstall dreamOS
```
sudo ./dk_uninstall.sh
```
